## 🧠 Data Structures & Algorithms in C
Beyond the fundamental syntax, I have dedicated a significant portion of this repository to **Data Structures and Algorithms (DSA)**. Implementing DSA in C is a unique challenge because it lacks built-in high-level libraries (like Python or Java), requiring me to:
- Manage memory manually using `malloc`, `calloc`, and `free`.
- Build complex structures like Linked Lists, Trees, and Graphs from scratch.
- Deeply understand how data is stored and accessed in RAM.

### Key DSA Implementations:
- **Linear Data Structures**: Arrays, Linked Lists (Singly, Doubly, Circular), Stacks, and Queues.
- **Non-Linear Data Structures**: Binary Trees, Binary Search Trees (BST), and Graphs.
- **Algorithms**: Searching (Linear/Binary), Sorting (Bubble, Merge, Quick, etc.), and Recursion.
